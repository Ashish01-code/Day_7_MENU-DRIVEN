# Day_7_MENU-DRIVEN
Title: Menu Driven List Operations in Python

Description:
This program demonstrates Python list operations using a menu-driven approach.
Operations included:
- Append
- Insert
- Remove
- Pop
- Length
- Sum
- Max & Min
- Sort (Ascending & Descending)
- Count occurrences

Concepts Used:
- Lists
- Loops
- Conditional statements
- Built-in functions
